# Captain Claw-Inspired 2D Platformer (LibGDX)

A small LibGDX project that captures the spirit of classic side-scrolling action:

- Side-scrolling camera.
- Jump/run platforming.
- Patrolling enemies.
- Collectible treasure.
- Win condition by reaching the ship area.
- Restart flow after win/lose.

## Controls

- **Move:** `A/D` or `Left/Right`
- **Jump:** `W`, `Up`, or `Space`
- **Restart:** `R` (after win/lose)

## Run

```bash
./gradlew :desktop:run
```

If you don't have a wrapper, use your local Gradle installation:

```bash
gradle :desktop:run
```
